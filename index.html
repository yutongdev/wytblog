[2024]Hexo部署Blog到Github

Node

npm install -g hexo-cli
npm install hexo-deployer-git –save

Git

hexo d // 上传到 github【需要先配置好 _config.yml 文件的上传地址】

Hexo

命令

hexo init ‘githubblog’ // 初始化项目
hexo deploy
hexo new [2024]Hexo部署Blog到Github // 创建博客
hexo s // 运行

配置

【_config.yml】
deploy:
type: git
repo: https://github.com/xxxxx/xxxxxxx.git # github address
branch: master
message: ‘Wyt blog’
